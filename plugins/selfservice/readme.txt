== Description ==

Links to a suitably configured SaltStack installation to allow users to create a
nd manage their own dedicated WordPress instances.

Part of a research infrastructure, which includes custom WordPress plugins as a key way of delivering functionality. Targetted at testing, trials and pilots, as well as longer-term deployments.

Work in Progress at the University of Nottingham, Mixed Reality Lab (in Computer Science) and Horizon Digital Economy Hub.


== Installation ==

1. Set up Salt Stack on the machine hosting WordPress - see something else for more details...
1. Upload this directory and all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a new Site (from the admin menu) and publish it
